# 隱私說明範本

> 本文件提供三種長度的隱私說明範本，承辦人可直接複製貼用。**使用前請確認你校內的資安政策實際情況**，本範本只是起點，不是法律建議。

---

## 範本 A：校內公告版（5 句話）

> 適用：張貼在校內公告欄、Line 群組、Email 群發。

```
【行政專案工作流隱私說明】

我們導入「學校行政專案工作流」協助整理公文、活動與成果資料。

本系統運作於本校自有 Google Workspace 內，所有資料留存於學校帳號，不外傳。
系統僅收集承辦人姓名與 Email 用於聯絡通知，不收集學生個資。
正式公告、經費核銷與成果送出仍由人工確認，AI 不替您決定。
若有疑問請洽 [承辦窗口姓名+分機]。
```

---

## 範本 B：給主任的書面說明（一頁版）

> 適用：附在簽呈或校務會議資料後面。

```markdown
# 「學校行政專案工作流」資料處理說明

## 1. 系統概述

本校導入開源工具「學校行政專案工作流」，用於協助行政承辦人從公文進件到成果送出
的全流程資料整理、期限追蹤與通知。

系統來源：MIT 開源，fork 自 mihozip/google-workspace-admin-project-workflow。

## 2. 資料儲存位置

- 所有資料儲存於本校 Google Workspace 帳號內的 Drive、Sheets、Docs、Calendar。
- 不使用任何外部伺服器、不外傳資料。
- 程式碼為 Google Apps Script，運行於 Google 提供的執行環境。

## 3. 系統處理的個資範圍

| 個資類型 | 是否處理 | 用途 |
|---|---|---|
| 承辦人姓名 | 是 | 寫入專案紀錄、通知信收件 |
| 承辦人 Email | 是 | 寄送系統通知 |
| 學生姓名 | 否 | 系統設計上不接觸學生資料 |
| 學生個資（學號、地址等） | 否 | 完全不接觸 |
| 家長個資 | 否 | 完全不接觸（除非承辦人手動上傳同意書，但同意書檔案本身位於 Drive，系統不解析其內容） |

## 4. 存取權限

- Drive 專案資料夾的存取權限**完全由承辦人自行設定**，系統不更動。
- 行政專案總控表預設僅系統管理者與資訊組可編輯。
- 表單回應試算表預設不公開，僅管理者可看。

## 5. 離職交接機制

- 若採用「共用雲端硬碟」（建議方式），承辦人離職後資料仍歸學校所有。
- 若採用個人帳號儲存，請於離職前將「Drive 資料夾擁有者」變更為交接人。

## 6. 不自動執行的事項（人工確認邊界）

本系統**不自動**執行下列事項，必須由人決定：

- 正式公告
- 經費核銷
- 成果送出
- 對外發文
- 法規/個資判斷
- 校內簽核

## 7. 與校內資安政策的對應

| 校內資安政策條文 | 本系統對應做法 |
|---|---|
| 不外傳學生個資 | 系統設計上不接觸學生資料 |
| 帳號權限分級 | 採用 Workspace 既有權限機制 |
| 重要資料離校備份 | Drive 已是雲端，符合異地備份原則 |
| 服務終止後資料保留 | reset 流程保留所有歷史資料 |

## 8. 風險評估

| 風險 | 機率 | 影響 | 緩解措施 |
|---|---|---|---|
| Google 服務中斷 | 低 | 中（資料安全但無法存取） | 重要資料定期匯出 |
| 承辦人帳號被盜 | 低 | 中 | 啟用雙重驗證 |
| 系統誤刪資料 | 低 | 高 | reset 流程不自動刪資料夾 |
| 公文機密性 | — | 視公文而定 | 機密公文不適用本系統，請依舊有紙本流程 |

## 9. 聯絡窗口

- 系統承辦：[姓名+分機]
- 資安問題：[資訊組分機]
```

## 範本 C：給家長的版本（如有需要）

> 適用：學校導入後，若家長詢問「我的孩子的資料會不會進入這個系統」。

```
親愛的家長您好：

本校近期導入「學校行政專案工作流」協助行政承辦人整理公文與活動資料。

向您說明三件事：

1. 本系統不收集、不處理學生個人資料。系統僅記錄行政承辦人的姓名與聯絡 Email，
   用於系統通知。

2. 您的孩子在校的學籍、成績、家庭資料等，仍由本校原有的校務系統管理，
   並未匯入本系統。

3. 若您曾為了某項活動填寫同意書，該紙本或電子檔可能會被承辦人上傳到本校
   Google Drive 的活動資料夾中歸檔。這份檔案的存取權限與校內既有政策相同。

若您對個資處理有任何疑問，請聯絡：
[承辦窗口] / [資訊組] / [校長室]

謝謝您。
[學校名稱] 敬上
```

---

## 使用建議

1. **先用範本 A** 在校內貼公告，讓老師知道有這個系統
2. **若主任、組長有疑問**，補上範本 B 一頁紙
3. **若家長有疑問再用範本 C**——絕大多數情況不會用到，因為系統本來就不碰學生資料

---

## 重要免責

本範本是**起點不是終點**。導入前請：

- 跟學校資訊組確認校內資安政策有沒有特殊條文
- 跟主任確認需不需要簽到主任會議或家長會
- 跟採用「共用雲端硬碟」的雲端管理者確認權限設定

若涉及法律解釋或個資爭議，請洽縣市教育局或專業法律顧問，本範本**不構成法律建議**。
